INT 21h AH=05H Printer Output (DOS 1.0+)
────────────────────────────────────────

Input:
    AH = 05h
    DL = Character to write

Output:
    None

Action:

    Sends the character in DL to standard printer device.

Notes:

    DOS checks for CTRL+C and CTRL+Break keys, and executes INT 23h if pressed.

    Normally standard printer is the first parallel port. Under DOS 2.0 and
    higher, standard printer may be redirected.

    If the printer is busy, this function will wait until it can send
    the character.

See also:

    INT 17H AH=00H WRITE CHARACTER
