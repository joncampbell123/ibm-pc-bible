All INT 31h functions, ordered by function call number (value of AX on entry).

INT 31h is the protected-mode entry point for the DOS Protected Mode Interface.

0.9+  AX=0003H  Get Next Selector Increment Value
0.9+  AX=0004H  Lock Selector ("Reserved" in later versions of the spec)
0.9+  AX=0005H  Unlock Selector ("Reserved" in later versions of the spec)
0.9+  AX=0006H  Get Segment Base Address
0.9+  AX=0007H  Set Segment Base Address

0.9+  AX=0009H  Set Descriptor Access Rights
