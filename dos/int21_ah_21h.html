INT 21h AH=21H Unused (DOS 1.0+)
────────────────────────────────────────────

Input:
    AH = 21h
    DS:DX = segment:offset of file control block (FCB)

Output:
    Return: AL = 0 (read successful)
	       = 01 if EOF (no data read)
	       = 02 if DTA is too small
	       = 03 if EOF (partial record read)

Action:
    Unused.
