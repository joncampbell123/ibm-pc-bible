All INT 2Fh functions, ordered by function call number (value of AH on entry).
Some functions have a subfunction (AL). If any other registers control the
function on entry, they are listed as well.

INT 2Fh carries many functions, from many parts of the system and from many
other utilities, as well as Windows.

Glossary:

ASSIGN.COM                  MS-DOS ASSIGN utility (to redirect a drive letter
                            to another drive)

DPMI                        DOS Protected Mode Interface

MSCDEX.EXE                  Microsoft CD-ROM extensions for MS-DOS

Netredir                    MS-DOS network redirector

PRINT.COM                   MS-DOS print spooler (PRINT.COM)

SAVE32.COM                  Windows 95 utility to preserve 32-bit registers
                            across invocations of hardware interrupt handlers

SHARE.EXE                   MS-DOS file locking/shared file access

Windows                     Microsoft Windows

Win/386 2.x                 Microsoft Windows 2.0 / 386

Windows 3.0                 Microsoft Windows 3.0

Windows 3.1                 Microsoft Windows 3.1

Windows 95                  Microsoft Windows 95

WINOLDAP                    Microsoft Windows WINOLDAP extension to support
                            "old" (character mode) application access to DDE,
                            menus, and the clipboard.
